# x-server-utils

FastAPI 服务器工具集与压力测试工具

## 安装

```bash
pip install x-server-utils
```

## 快速开始

python

```
from x_server_utils import core

# 使用示例（根据你的实际功能调整）
```



## 功能

- FastAPI 服务器工具函数
- HTTP 压力测试
- 日志管理（基于 loguru）

## 依赖

- Python >= 3.11
- fastapi
- uvicorn
- requests
- loguru
